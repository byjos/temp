# github 使用
## 在本地环境建一个库
* 1、git init
* 2、git remote add origin https://github.com/byjos/temp.git

* 3、git add temp
将本地文件提交上去
* 4、git config --global user.email "you@example.com"
* 5、git commit -m "temp"
* 6、git push -u origin master 
* 在ip没有识别的情况下要去coding上生成ssh key
* ssh-keygen -t rsa -C "541060364@qq.com"
* 生成密钥，放在coding上面。
